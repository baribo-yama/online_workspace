# 障害物画像ガイド

## 推奨設定

### 画像仕様
- **解像度**: 32x32 または 48x48 ピクセル
- **ファイル形式**: PNG8 (256色) または WebP
- **ファイルサイズ**: 5KB以下推奨
- **背景**: 透明推奨

### 最適化方法

#### オンラインツール (無料)
1. **TinyPNG** (https://tinypng.com/)
   - PNG画像の圧縮に特化
   - 透明度を保持

2. **Squoosh** (https://squoosh.app/)
   - Googleが開発した画像最適化ツール
   - WebP形式への変換も可能

#### ソフトウェア
1. **GIMP** (無料)
   - エクスポート時に「PNG8」を選択
   - 「色数を256色に減らす」

2. **Photoshop**
   - Web用に保存 → PNG-8
   - 色数を64-256色に設定

### ピクセルアート風にする方法

1. **解像度を下げる**: 16x16 → 48x48に拡大
2. **色数を制限**: 16-64色程度
3. **アンチエイリアスを無効**: くっきりとしたエッジ

## ファイル名

- `obstacle1.png` - 1番目の障害物
- `obstacle2.png` - 2番目の障害物  
- `obstacle3.png` - 3番目の障害物

## 例: 8ビット風キャラクター

```
推奨サイズ: 32x32ピクセル
色数: 16色以下
ファイルサイズ: 2-3KB
```

## テスト用サンプル

シンプルな図形から始めることをお勧めします：
- 四角形、円形、三角形
- 2-3色のシンプルなデザイン
- グラデーションは避ける
