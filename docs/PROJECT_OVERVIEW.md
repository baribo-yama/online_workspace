# PROJECT_OVERVIEW — MOKU

## Mission
MOKUは、オンライン上で仲間と集まり、集中作業や学習を行える仮想ワークスペース（オンライン実習室）を提供するアプリケーション。  
ユーザーは「部屋」を作成または選択して入室し、リアルタイムのビデオ通話で交流できる。  
さらに、集中を支援するポモドーロタイマーや、休憩中のエンタメ要素を備え、生産性とモチベーションの両立を目指す。

---

## 対象ユーザー
- リモートで作業を行う学生・チーム  
- オンライン自習・共同学習グループ  
- 趣味・学習コミュニティ  
- 大学や教育機関（例：Tech Garageなど）

---

## Core Value
| 機能 | 価値 |
|------|------|
| ログイン不要の即入室設計 | 「気軽に集まれる」オンライン作業空間を実現 |
| リアルタイム同期ポモドーロタイマー | グループ全員で集中時間を共有し、生産性を高める |
| LiveKitベースの高品質ビデオ通話 | 滑らかな映像と低遅延で快適なコミュニケーション |
| チャット機能（予定） | 作業・休憩中を問わずテキスト交流が可能 |
| エンタメ機能（休憩限定） | 作業の合間に短時間でリフレッシュできる体験 |

---

## 現在の進捗（MVP完了）
MVP開発は完了しており、以下の主要機能が安定動作している。

- 部屋の作成、一覧、入室  
- 複数人でのビデオ通話（LiveKit利用）  
- 共有ポモドーロタイマー  
- 障害物よけゲーム（動作は確認済みだが、処理負荷のため使用停止中。コードは保持）

今後は **バグ修正と機能追加、保守性の向上** に重点を置くフェーズへ移行している。
