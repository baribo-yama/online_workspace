# PROJECT_OVERVIEW — MOKU

## Mission
MOKUは、オンライン上で仲間と集まり、集中作業や学習を行える仮想ワークスペース（オンライン実習室）を提供するアプリケーション。  
ユーザーは「部屋」を作成または選択して入室し、リアルタイムのビデオ通話で交流できる。  
さらに、集中を支援するポモドーロタイマーや、休憩中のエンタメ要素を備え、生産性とモチベーションの両立を目指す。

---

## 対象ユーザー
- リモートで作業を行う学生・チーム  
- オンライン自習・共同学習グループ  
- 趣味・学習コミュニティ  
- 大学や教育機関（例：Tech Garageなど）

---

## Core Value
| 機能 | 価値 |
|------|------|
| ログイン不要の即入室設計 | 「気軽に集まれる」オンライン作業空間を実現 |
| リアルタイム同期ポモドーロタイマー | グループ全員で集中時間を共有し、生産性を高める |
| LiveKitベースの高品質ビデオ通話 | 滑らかな映像と低遅延で快適なコミュニケーション |
| チャット機能（予定） | 作業・休憩中を問わずテキスト交流が可能 |
| エンタメ機能（休憩限定） | 作業の合間に短時間でリフレッシュできる体験 |

## 優先度別追加予定機能

### **高優先度（Must）**
| 機能 | 概要 | 目的・補足 |
|------|------|------------|
| チャット機能 | ルーム内で参加者全員が利用できるテキストチャット | コミュニケーション強化、UIはLINE風のシンプル構成を想定 |
| 背景ぼかし機能 | 通話時に背景を自動的にぼかす | プライバシー保護、実習・授業用途を意識 |
| カメラ・マイク設定画面 | ホーム画面でオン／オフを事前設定可能にする | 入室時の自動オン問題を解消、ユーザー体験向上 |

### **中優先度（Should）**
| 機能 | 概要 | 目的・補足 |
|------|------|------------|
| ホスト権限の受け渡し | ルーム内の管理権限を別ユーザーに委譲 | 長時間運用時の安定化 |
| ルーム参加人数上限設定 | ルーム作成時に人数制限を設定可能に | 大規模利用時の負荷対策、将来的に設定値を定数化 |

## 大学テスト導入までに完了させたい項目

| タスク | 目的 |
|--------|------|
| 重大バグの修正 | 特に人数制限ロジック、参加者リスト未更新の2件を優先対応 |
| チャット機能の追加 | コミュニケーション向上、正式導入前の必須項目 |
| 入室前のカメラ／マイク設定追加 | プライバシー・UX両面で改善 |
| ランダム話題ボード追加 | 休憩時間中の交流促進、既存のポモドーロとの連携 |
| 一定時間無操作時の自動ルーム終了 | 管理コスト削減、サーバー負荷の軽減 |
